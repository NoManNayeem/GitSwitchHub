Get Started in Minutes

Follow this simple tutorial to set up GitSwitchHub

1

Download & Install

Download the latest release from GitHub and install the .dmg file on your macOS system.

# Download from GitHub Releases
2

Add Your GitHub Accounts

Open GitSwitchHub and add your GitHub accounts using personal access tokens.

# Create token at github.com/settings/tokens
3

Map Your Repositories

Map specific repositories to specific accounts for automatic switching.

# Repository URL → Account mapping
4

Install Git Helper

Install the Git credential helper to enable automatic account switching.

# One-click installation in Settings
